# waze-rs
Calculate actual route time and distance with Waze API - based on Python [WazeRouteCalculator](https://github.com/kovacsbalu/WazeRouteCalculator)

Uses serde and reqwest to make requests to Waze API.

## Build
```bash
cargo build
```
